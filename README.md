# de-densification

Fall 2025 and spring 2026 course data were scraped from the CU Directory of Classes on Dec. 7 at approximately 10 p.m. Generative AI tools were utilized in the creation of the web-scraping algorithm. 

Spring 2026 course data was scraped from the Columbia Directory of Classes on Dec. 7 at approximately 10 p.m. Data on full or overenrolled classes across all departments was observed. Generative AI tools were utilized in the creation of the web-scraping algorithm.
